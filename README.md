# dempsey-center-analysis
CS5100-2026spring-Final project for Dempsey Center client data analysis

## Overview
This project analyzes Dempsey Center client survey data from 2022 to 2025 using exploratory data analysis, PCA, and clustering.

## Repository Structure
- `eda_analysis.ipynb`: exploratory data analysis and summary statistics
- `requirements.txt`: Python package dependencies
- `5100 Dempsey Center Report`: Slides

## How to Run
1. Install dependencies:
   `pip install -r requirements.txt`
2. Open `eda_analysis.ipynb`
3. Run all cells from top to bottom
